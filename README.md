A full-stack Pakistani Heritage Recipe application.

---

## 📁 Projects

### 🎨 Project 1 — Frontend Interface
📂 Folder: `project-1-frontend/`
Pure HTML, CSS, JavaScript. Responsive, mobile-first.
- Open `index.html` in any browser to run it

### ⚙️ Project 2 — Backend REST API
📂 Folder: `project-2-backend/`
Node.js + Express. RESTful endpoints, validation, JSON.
- Run `npm install` then `npm start` to launch the server

### 🗄️ Project 3 — Database Integration
📂 Folder: `project-3-database/`
Node.js + Express + Supabase (PostgreSQL). Full CRUD with real cloud database.
- Run `npm install` then `npm start` to launch the server
- Requires `.env` file with Supabase credentials

## 🚀 How to Run

**Project 1 — Frontend**
Open `project-1-frontend/index.html` in any browser.

**Project 2 — Backend**

```bash
npm install
npm start
```

**Project 3 — Database**
```bash
cd project-3-database
npm install
```
